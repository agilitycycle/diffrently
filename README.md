## Run locally

**Note: Must set up Firebase acc. first.**

Clone the project

```bash
  git clone https://github.com/jamesstar89/flipbio-app
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
